# Documentacion Canonica (Activa)

Este directorio es la fuente de verdad operativa para el proyecto.

## Archivos activos

- `../phpmyadminbonifaciostablas.sql`: export de referencia MySQL (phpMyAdmin) con nombres de tablas y columnas; usar al escribir `INSERT`/`UPDATE` en la API.
- `SYSTEM-ARCHITECTURE.md`: arquitectura actual, flujos y componentes.
- `OPERATIONS-RUNBOOK.md`: despliegue, sync, verificaciones y recuperacion.
- `ERROR-FIXES-LOG.md`: registro vivo de incidentes, causa raiz y fix.
- `backlog.md`: tickets de trabajo y estado actual.
- `requerimientos-nuevos.md`: requerimientos recientes (mapa admin, layout global, reuniones/Jitsi) con IDs y criterios de aceptación.

## Archivos historicos (consulta)

Los siguientes archivos NO se eliminan y se conservan como referencia historica:

- `../documentacion.md`
- `../errores.md`

Regla: para trabajo diario usar `docs/*`; usar historicos solo para contexto o casos similares.
