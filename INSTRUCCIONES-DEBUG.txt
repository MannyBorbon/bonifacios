PASOS PARA DIAGNOSTICAR EL ERROR 500 EN SUBMIT.PHP
=====================================================

1. SUBIR ARCHIVOS AL SERVIDOR (vía Hostinger File Manager):
   
   Sube estos archivos a: /public_html/api/applications/
   
   ✓ submit-debug.php
   ✓ submit-minimal.php
   
   (Ya subiste test-submit.php, ese está bien)

2. ABRIR ARCHIVO DE PRUEBA EN TU NAVEGADOR:
   
   - Busca el archivo: test-submit-form.html
   - Haz DOBLE CLIC para abrirlo en tu navegador
   - NO lo subas al servidor, ábrelo localmente

3. HACER LAS PRUEBAS:
   
   En la página que se abre, verás 3 botones:
   
   a) Haz clic en: "Probar submit-minimal.php"
      → Esto prueba con solo 4 campos
      → Copia el resultado que aparezca
   
   b) Si el anterior funciona, haz clic en: "Probar submit-debug.php"
      → Esto prueba con todos los campos + error reporting
      → Copia el resultado completo que aparezca

4. COMPARTIR RESULTADOS:
   
   Copia y pega aquí TODO el texto que aparezca en la página,
   especialmente si dice "ERROR" o muestra un stack trace.
